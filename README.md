# Assignment Tracker (Desktop)

A Tauri + React desktop app to track college assignments.

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the desktop app in dev mode

```bash
npm run tauri dev
```

## Notes

- Bundling is disabled for now to avoid missing icon assets. When you are ready to package the app, enable bundling and add the required icons in `src-tauri/icons`.
